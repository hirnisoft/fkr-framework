# Functional Complexity Reduction (FKR)

## Definition

Functional Complexity Reduction (FKR) is an approach to designing everyday life, systems, and decisions in which unnecessary complexity is removed and necessary complexity is clearly structured.

The goal is maximum clarity with minimal mental load.

---

## Core Principle

> Complexity is not avoided, but consciously designed.

---

## Core Idea

FKR does not start with behavior, but with the system.

Not:
- “How can I work more efficiently?”
- “How can I focus better?”

But:

> “Why does this complexity exist at all?”

---

## Scope

FKR applies to all types of systems, including:

- things you own  
- spaces you use  
- processes you repeat  
- decisions you make  

Everything can be either clear or complex.

---

## Summary

Functional Complexity Reduction (FKR) is an approach to designing systems and everyday life by removing unnecessary complexity and structuring what remains.

The result is maximum clarity with minimal mental load.
